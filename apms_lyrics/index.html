APMS Lyrics

노래 가사가 부드럽게 따라오는 Lyric Video를 쉽게 만들기 위한 After Effects 플러그인입니다.
Source Code

Installation

0. After Effects version must be higher than or equal to 18.0

1. Download the latest release

2. Put the extension folder in
Win: C:\Users\<USERNAME>\AppData\Roaming\Adobe\CEP/extensions
macOS: ~/Library/Application Support/Adobe/CEP/extensions

3. Run AE and click Windows -> Extensions -> APMS Lyrics to open the extension.

Usage

1. Create a composition for lyrics

2. Add a text layer with lyrics
* Lines with only "$I" are replaced with intervals.

3. Set text style

4. Select the composition and hit 'Make' button

5. Adjust markers' timing to match the lyrics and the song

배경

노래를 만들었을 때, 대중에게 쉽게 공유하기 위해서 유튜브에 노래를 올리곤 합니다. 그 노래에 가사가 있다면, 검은 화면에 노래만 있는 것 보다는 lyric video를 만드는 것이 좋은 생각일 것입니다. 타이포그래피와 모션 그래픽으로 이루어진 화려한 것이 아니어도 됩니다.

Apple Music Lyrics Animation 예시

Apple Music의 synced lyrics의 모습입니다. 가사 줄이 한번에 이동하지 않고, 뒤에 있을수록 딜레이를 가지고 이동합니다. 신호 대기 중이었던 자동차들이 신호가 바뀌고 차례대로 출발하는 모습과 비슷합니다.

또한 highlight되는 가사 외에는 blur와 투명도가 들어갑니다.

저는 Apple Music과 비슷한 스타일의 lyric video를 만들고 싶었습니다. 하지만 각 타이밍별로 키프레임을 복사하고, 각 가사 줄마다 지연을 주며 위치를 바꾸는 것은 수작업으로 하기 힘든 일이었습니다. 그래서 이것을 쉽게 해줄 플러그인을 만들고 싶었습니다.

Adobe After Effects의 plugin을 만드는 것에는 두 가지 방법이 있습니다. Adobe SDK와 C++을 이용하거나, CEP JavaScript libraries와 HTML/CSS/JS, ExtendScript를 이용하는 방법입니다. 이 플러그인의 제작에는 후자가 사용되었습니다.

이 방법으로는 플러그인의 Panel을 HTML/CSS/JS로 만들고, 내부 로직은 ExtendScript로 작성하며 이 둘을 CEP library로 이어줍니다. ExtendScript는 옛날 ECMAScript 표준에서 파생되어 Adobe 프로그램을 조작하는 script를 작성할 때 사용하는 언어입니다. ExtendScript가 너무 옛날 JavaScript를 기반으로 하고 있어서 불편한 점이 많이 있었습니다. 예를 들어, function.bind method가 존재하지 않아 function.call을 대신 사용하는 방식으로 바꿔야 하기도 했습니다.
