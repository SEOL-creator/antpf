Everygimpo (Timetable)

고등학교 시간표 정보, 급식 정보 등을 제공하는 웹사이트/어플리케이션(PWA)입니다.
같은 반 학생끼리도 이동 수업, ABCD 타임 수업이 있어 시간표가 나뉘는데, 이를 개인별로 맞추어 시간표를 제공합니다.
특정 교시의 수업이 바뀌거나, 일 단위로 수업이 다른 날과 교환되는 경우도 간단히 처리할 수 있습니다.
또한 시간표(각 교시의 수업을 지정함)와 시정표(각 교시의 시작/종료시간을 지정함)를 분리하여 코로나 시기 빈번했던 수업 시간의 변경에도 쉽게 대응할 수 있도록 하였습니다.

배경

고등학교 1학년 2학기 정보 시간에 파이썬으로 프로그래밍 언어를 처음 접하고, 뭐든지 만들 수 있을 것 같다는 생각이 들어 기존에 불편했던 시간표 어플리케이션들을 대체하는 것을 만들어보자는 생각에 제작한 것입니다.

고등학교 입학과 동시에 코로나 사태가 발생하였고, 원격 수업과 대면 수업을 수시로 오가며, 시간표와 시정표의 변동이 빈번하였습니다. 특히 시정표의 변동은 기존의 거의 모든 시간표 어플리케이션에서 간단히 수정하기 어려웠습니다.

그래서 멋지고, 세련되고, 쓸만한 것을 만들어 보고 싶어서, 웹 개발 방법을 학습하며 만들기 시작했습니다. Backend server application으로 Django를 선택한 것은 단지 알고 있던 Programming Language가 Python이었기 때문이었습니다.

처음에는 웹서버의 역할이 무엇인지, Backend와 Frontend의 구분은 무엇인지조차 알지 못했지만 기본적인 웹/CS 정보를 알려주는 블로그, 그리고 ReactJS, Django 등을 쉽게 가르쳐주었던 공식 Documentation을 학습하여 지금은 원하는 것을 쉽게 구현할 수 있게 되었습니다.
